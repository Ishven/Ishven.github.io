如何使用hexo搭建个人博客
发表于 2020-05-31| 更新于 2020-05-31|工具
阅读量:

使用hexo

  • 要有nodejs环境

  1. 全局安装博客脚手架

    npm install -g hexo-cli

  2. 创建一个目录，初始化一个hexo项目

    hexo init

  3. 启动项目

    hexo s

  • 清理项目命令

    hexo clean

  • 生成命令

    hexo g

  • 安装发布工具

    npm install –save hexo-deployer-git

配置_config.yml文件

  • 将项目部署到发布地址命令

    hexo deploy

  • 写博客

    E:\blog\source_posts

爬取哔哩哔哩

hexo bangumi -u

创建页面

hexo new page photos
