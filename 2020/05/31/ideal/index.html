ideal使用教程
发表于 2020-05-31| 更新于 2020-05-31|工具
阅读量:

快捷键

ideal设置

  • 打开设置：ctr + alt + s

查看代码

  • 回退（撤消最后导航操作） ：Ctrl+alt+left、right
  • 显示当前类的变量和方法：ctrl+f12
  • 查看方法实现：ctrl+alt+鼠标左键
  • 查找类：ctrl+n
  • 全局查找：Ctrl+Shift+n
  • 显示类的继承层次 (接口)：Ctrl+H
  • 查找类/变量在工作区域工程中使用 ： Alt+F7

写代码

  • 代码补全：ctrl+shift+enter
  • 代码建议：ctrl+alt+v，alt+enter
  • Ctrl+O：重写方法
  • Ctrl+D，复制行
  • for、if补全：ctrl+j
  • 格式化代码：Ctrl+Alt+L
  • 导包： Ctrl+Alt+O
  • 向下插入一行：shift+enter

调试

  • F9 恢复程序
  • Alt+F10 显示执行断点
  • F7 单步运行（进入方法）
  • Shift+F8 跳出方法
  • F8 单步运行

ideal好用的插件

  • Free Mybatis plugin
  • Lombok

ideal开发vue配置

安装插件后重启，打开File→setting→Editor→File Types 找到vue.js template

那就弄好了vue的模版文件了，接下来只需要生成这文件就可以了
